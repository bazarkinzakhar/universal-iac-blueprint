# Универсальный шаблон IaC (Terraform + Ansible)

Production ready iac шаблон. Репозиторий предлагает масштабируемый, безопасный и независимый от провайдера подход (текущая конфигурация под AWS) для развертывания и настройки облачных ресурсов

## Ключевые особенности

- **Модульный Terraform:** Чистая архитектура с разделением на модули `vpc` и `compute`
- **Удаленное хранение стейта (Remote State):** Настроено использование S3 в качестве бэкенда с блокировками через DynamoDB
- **Динамический инвентарь:** Ansible автоматически находит инстансы через API облачного провайдера
- **Безопасность прежде всего:** Строгие группы безопасности (Security Groups), усиление защиты SSH (hardening) и управление секретами
- **Идемпотентность:** Переиспользуемые роли Ansible (`base_system`, `docker`, `observability`)

## стек

- **Provisioning:** Terraform >= 1.5.0
- **Configuration:** Ansible >= 2.15
- **Cloud Provider:** AWS (совместимо с Free-Tier)
- **CI/CD:** GitHub Actions (Линтинг, TF Plan)

## порядок запуска

1.  **Установить зависимости:**
    `brew install terraform ansible awscli`
2.  **Настроить учетные данные облака:**
    `aws configure`
3.  **Развернуть инфраструктуру:**
    ```bash
    make init ENV=stage
    make apply ENV=stage
    ```
